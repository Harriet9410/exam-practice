# 📚 刷题练习

一个纯前端的在线刷题网站，支持多科目、多题型、云端同步，无需注册即可使用。

**在线访问：[https://harriet9410.github.io/exam-practice/](https://harriet9410.github.io/exam-practice/)**

## 功能特性

### 题目练习
- **5 种题型**：单选题、多选题、判断题、填空题、问答题
- **科目/章节分类**：左侧边栏按科目 > 章节两级分组，可折叠展开
- **搜索过滤**：按关键词搜索题目，按掌握状态筛选（全部/未掌握/已掌握/收藏/错题）
- **随机打乱**：一键打乱题目顺序，选项同步随机
- **快捷键**：← → 切换题目，Space 显示答案，S 收藏，K 标记掌握

### 进度管理
- **标记掌握/错题/收藏**：每道题可标记状态，方便针对性复习
- **本地存储**：进度自动保存在浏览器 localStorage，刷新不丢失
- **重置进度**：一键清除所有学习记录

### 导入题目
- **文件导入**：支持 TXT（GBK 自动检测）、DOCX（mammoth.js）、PDF（pdf.js）
- **文本粘贴**：直接粘贴题目文本，自动解析
- **追加导入**：新导入的题目追加到现有列表，不会覆盖

### 云端同步
- 自动上传/下载题目数据到 npoint.io
- 每次修改自动同步，跨设备共享题目和答案
- 个人学习进度（掌握/错题/收藏）独立存储在本地

### 批量编辑
- 📚 批量修改科目
- 🏷️ 批量修改章节分类
- 🗑️ 批量删除题目
- ✏️ 编辑/添加答案

### 其他
- 🌙 深色模式切换
- 💾 导出题目为 JSON
- 带公式的物理题以截图形式展示，保证公式完整显示

## 内置题库

| 科目 | 题数 | 章节 |
|------|------|------|
| Java | 19 | 基础知识、编程题 |
| 毛概 | 344 | 单选、多选、判断 |
| 大学物理 | 52 | 第1-7章选择题 |

共计 **415** 道题，全部含参考答案。

## 技术实现

- 纯静态网站，无需后端服务器
- 单页应用，零依赖框架
- 云端同步基于 [npoint.io](https://npoint.io) API
- DOCX 解析：[mammoth.js](https://github.com/mwilliamson/mammoth.js)
- PDF 解析：[pdf.js](https://github.com/nicbarker/pdf.js)

## 文件结构

```
├── index.html      # 页面结构
├── style.css       # 样式（含深色模式）
├── app.js          # 核心逻辑
└── questions.js    # 题库数据（415道）
```

## 本地运行

```bash
# 安装 http-server
npm install -g http-server

# 启动
http-server -p 8080 -c-1
```

浏览器访问 `http://localhost:8080` 即可。
